## Overview

A compact Git‑like VCS in C++17 with colored logs and multi‑commit traversal.

### Highlights

- SHA‑1 object storage
- `.gudignore` + scoped repo ops
- `gud echo` (diff) with line‑level plans

```bash
# demo
$ gud create && gud add . && gud commit -m "first commit"
```
